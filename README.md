# Habla Juega

> Duolingo for Spanish speech therapy practice.

See [Project.md](./Project.md) for the full project brief, architecture, and roadmap.

## Branch model

- `main` — production. Auto-deploys to Vercel.
- `dev` — integration. All work lands here first via feature branches; PRs from `dev` to `main` cut releases.

## Quick start

```bash
pnpm install
pnpm dev
```

Requires Node 20+ and pnpm 9+.

## Status

Pre-development. Phase 0 in progress.
